## Keyboard related

- Assign Close Window to Command + W

### Swap Command and Control keys

To swap Command and Control -> https://askubuntu.com/a/137456

~~~bash
vim ~/.xmodmap

remove mod4 = Super_L
add control = Super_L

xmodmap .xmodmap
# https://askubuntu.com/a/1082273 Dont add it to .zshrc
~~~
